# Regular Expression

## Questions

How to use regex match with a variables?
:   Use `RegExp` constructor

    ```js
    // multiple flags at once, global and ignore-case
    const pattern = new RegExp(myVar, 'gi');
    myBlob.match(pattern)
    ```
