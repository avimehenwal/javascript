# :: Landmarks in js development

year Description
1995 Brendan Eich (opens new window) made javascript at netscape
1996 ECMA-262 specification in 1996, under a technical committee at ECMA known as TC39 (opens new window).
Sun wouldn’t transfer ownership of the JavaScript trademark to ECMA, and while Microsoft offered JScript, other member companies didn’t want to use that name, so ECMAScript stuck.
1997 The first edition of ECMA-262 was released
2003 Mozilla Foundation was formed as AOL fired 50% of netscape employees
TC39 (opens new window) was disbanded
2009 the fifth edition of ECMAScript was published
2015 formalize ECMAScript 6
became a rolling stone
2015 Babel transpiler project (opens new window)
ES2017 asynchronous loading of javascript modules using import()
